Read the following before using the files within this archive.

1. This archive contains files that belong to the articles at: 
http://www.adobe.com/devnet/flash/articles/video_gallery.html
http://www.adobe.com/devnet/flash/articles/vidgal_structure.html

Use these files with the article as a template for creating an ActionScript 3.0 Flash video gallery.

2. Unpack the archive and put the files in the following locations:

* Extract the Flash Video Gallery AS3.0 folder to your desktop or any location on your hard drive.
* Open the supplied project file (FlashVideoGallery.flp) in Flash CS3 Professional to view the project.
* Open the ActionScript 3.0 files in the fvg folder and the FlashVideoGallery.fla file in Flash CS3 Professional to edit.
