One drawback of the private members when used with constructors is that they are recreated every time the constructor is invoked to create a new object.

This is actually a problem with any members you add to this inside of constructors. In order to avoid this duplication of effort and save memory, you can add common properties and methods to the pro- totype property of the constructor. This way they are shared among instances created with the same constructor.
